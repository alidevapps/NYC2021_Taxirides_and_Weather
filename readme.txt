This repository contains the python code and downloaded data for the Course Data Wrangling for the Bachelor Business Analytics. 

Before running the code you need an app token from the New York open data repository at https://opendata.cityofnewyork.us/


With this app token make a text file 'configuration.txt' in your working directory with the two following lines. Do not remove the text 'app_token: '
----
[ny_opendata]
app_token: the app token you generated

----
See https://dev.socrata.com/docs/authentication.html for more information on the app token
